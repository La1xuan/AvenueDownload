# AvenueDownload

可以下载Avenue上的视频，以便去图书馆等地方自习。

## Installation

有数个版本，但都需要最新版本的python。  
json版本：会自主搜索指定位置的文件，读取json表并获取下载链接，预计规模和命名  
控制台版本：可在控制台直接输入下载链接，规模和命名  
pyautogui版本：提供一个输入框  
  
三个版本都在，自行改comment  

## USAGE
需要打开avenue上需要下载的视频的页面，F12打开网络控制台-打开网络标签页-播放视频-找到seg-1开头的选项-复制值，这个值需要填入网址栏。
